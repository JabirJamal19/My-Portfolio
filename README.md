# Personal Portfolio Website

This is a fully responsive personal portfolio website developed using **HTML**, **CSS**, **JavaScript**, and **PHP**. It showcases my skills, projects, resume, and contact information, aimed at highlighting my professional profile to potential employers and clients.


## 🚀 Features

- Clean and modern UI
- Responsive design for all devices
- Project showcase with links
- About Me and Resume section
- Contact form with PHP backend
- Interactive and smooth animations

- **Backend**: PHP
